abstract shared @property TimestampFormat  timestampFormat();

Returns actual time stamp representation format used in server.

Note:
This property tells particular HAVE_INT64_TIMESTAMP version flag that is used by remote server.
Note:
Will fallback to Int64 value if server protocol doesn't support acquiring of 'integer_datetimes' parameter.

abstract shared @property immutable(TimeZone)  timeZone();

Returns server time zone. This value is important to handle time stamps with time zone specified as libpq doesn't send the information with time stamp.

Note:
Will fallback to UTC value if server protocol doesn't support acquiring of 'TimeZone' parameter or server returns invalid time zone name.
